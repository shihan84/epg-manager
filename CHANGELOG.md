# Changelog

All notable changes to EPG Manager will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added

- Initial release of EPG Manager
- Channel management with multi-language support
- Program management with categories and thumbnails
- Schedule management with bulk operations
- EPG generation in XMLTV, JSON, and M3U formats
- Multi-language support for 10+ Indian languages
- User authentication and role-based access control
- RESTful API for external integrations
- Hosted EPG URLs for distribution
- Dashboard with real-time statistics
- Schedule templates for common programming patterns
- Bulk scheduling for daily, weekly, and monthly patterns
- Content management system with organized navigation
- Professional footer with company information
- Comprehensive documentation and legal notices

### Features

- **Channel Management**: Create and manage TV channels with logos, descriptions, and streaming URLs
- **Program Management**: Add programs with categories, thumbnails, and metadata
- **Schedule Management**: Create program schedules with flexible timing and repeat options
- **EPG Generation**: Generate EPG files in multiple formats (XMLTV, JSON, M3U)
- **Multi-language Support**: Support for Hindi, Tamil, Telugu, Bengali, Gujarati, Kannada, Malayalam, Marathi, Punjabi, Urdu, and more
- **API Access**: RESTful API for integration with external systems
- **User Management**: Role-based access control (Admin/Client)
- **Real-time Statistics**: Dashboard with usage metrics and content health
- **Bulk Operations**: Create multiple schedules at once
- **Template System**: Pre-built templates for common programming patterns

### Technical Details

- Built with Next.js 15 and React 18
- TypeScript for type safety
- Tailwind CSS for styling
- Radix UI for components
- SQLite database with Prisma ORM
- NextAuth.js for authentication
- Lucide React for icons
